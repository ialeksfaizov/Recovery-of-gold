# Предсказание коэффициента восстановления золота из золотосодержащей руды

## Описание проекта
Модель должна предсказать коэффициент восстановления золота из золотосодержащей руды для условной золотодобывающей компании.


## Stack
- Pandas
- Numpy
- Scikit-learn 
- SciPy
- Matplotlib
- Seaborn
